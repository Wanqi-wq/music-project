# vue音乐播放器

## 功能
### 1.包含首页，歌手，排行，搜索等多个页面
### 2.在播放页中，实现播放模式的选择，随机播放，顺序播放和单曲循环;歌词随歌曲播放而滚动
### 3.拖动进度条，歌曲播放进度发生变化；上一曲，下一曲切换，展示当前播放列表，删除和添加歌曲进当前播放列表中
### 4.在我的页面中，实现了歌曲收藏功能
